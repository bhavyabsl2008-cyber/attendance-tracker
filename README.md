# Attendance Tracker — Chitkara University

A smart attendance tracker for Chitkara University students. 
More useful than Chalkpad — tells you what Chalkpad doesn't.

## Features
- Track attendance per subject
- See how many classes you can safely miss
- DL/ML leave simulator
- End semester prediction
- Color coded warnings

## Live App
[Open Attendance Tracker](https://bhavyabsl2008-cyber.github.io/attendance-tracker/)

## Built With
HTML, CSS, JavaScript — no frameworks, no backend
