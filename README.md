# Attendance Tracker — Chitkara University

> Smarter than Chalkpad. Built by students, for students.

A web app that tells you what your university portal doesn't —
how many classes you can skip, what happens if you take leave,
and whether you'll make it to exams.

## Live App
[Open Attendance Tracker](https://bhavyabsl2008-cyber.github.io/attendance-tracker/)

## Features
- Track attendance per subject with color coded warnings
- Smart alerts — "you can safely miss 6 more classes"
- Full Day DL / Partial DL / Medical Leave simulator
- Skip next / Attend next live predictor
- End semester best case and worst case prediction
- Attendance chart with 75% threshold line
- Export, Import and Share your data
- Works on mobile
- No login, no backend — everything saved in your browser

## Built With
HTML · CSS · JavaScript — no frameworks, no backend
